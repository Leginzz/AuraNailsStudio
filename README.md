# 🚀 Cómo ejecutar el proyecto

1. Clonar el repositorio

```bash
git clone https://github.com/Leginzz/AuraNailsStudio.git
```

2. Entrar al proyecto

```bash
cd AuraNailsStudio
```

3. Abrir el archivo

```text
index.html
```

No requiere instalación de dependencias.
