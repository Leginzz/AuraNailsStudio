# Aura Nails Studio

Sitio web oficial de **Aura Nails Studio** — Salón de uñas en Tuxpan, Veracruz.

🌐 [leginzz.github.io/AuraNailsStudio](https://leginzz.github.io/AuraNailsStudio)

## Tecnologías

- HTML + CSS + JS (sitio estático)
- Google Fonts (Cormorant Garamond + Poppins)
- JSON-LD para datos estructurados (LocalBusiness)
- Diseño responsive, accesible, con modo oscuro
- Optimizado para GitHub Pages

## Design Lab

En `src/design-lab/` encontrarás 10 propuestas de diseño independientes en React + Vite + TypeScript. Cada una con personalidad, layout y experiencia de usuario únicos.

```bash
cd src/design-lab
npm install
npm run dev
```
